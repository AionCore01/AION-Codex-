## Despliegue
1) Subí los archivos a la rama publicada (root o /docs).
2) Settings → Pages → Deploy from a branch → main/(root|/docs).
3) Esperá 1–2 min y abrí: https://TU-USUARIO.github.io/NOMBRE-REPO
